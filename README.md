# Sample Hardhat Project

Try running some of the following tasks:

```shell
yarn hardhat help

yarn install
yarn hardhat compile
yarn hardhat node
yarn hardhat run scripts/deploy.js --network localhost
yarn hardhat test
yarn hardhat coverage

REPORT_GAS=true npx hardhat test
```
